# PyPerlinNoise - Cavegenerator for AntSim
Randomly creates a Cave with:
- Walls (grey)
- Food-Sources (Green)
    - not colliding with walls
    - not nearby of Ant-Nest (white)
- Nest (white)
    - checks for walls
